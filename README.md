CA Project
=========


Development
-------------

### Environment


### Setup
cd into project
`run npm install`

### Run Server
See project environment repo

### Run node server 
From within your directory
`npm start`
For production
`npm run prod`

### Lint

With every commit the airbnb linter is run to check for errors and to check that code styling follows the airbnb guidelines.
To run the linter you can also do: 
`npm run lint-all`
